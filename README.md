# Matrix Calculator in C

A fully functional **Matrix Calculator** written in C that supports
dynamic memory allocation and performs a wide range of matrix operations
including addition, subtraction, multiplication, scalar operations,
transpose, determinant, and inverse using **Gauss-Jordan elimination**.
